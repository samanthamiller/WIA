EasyTabs.js

What is EasyTabs.js?

EasyTabs.js is an easy to use jQuery plugin that provides tab functionality, while leaving the styling up to the designer.

How to use EasyTabs.js

EasyTabs.js HTML template

 
						
<html>
<head>
  <title>EasyTabs.js</title>
  <script src="js/jquery-1.7.1.min.js" type="text/javascript"></script>
  <script src="js/jquery.hashchange.min.js" type="text/javascript"></script>
  <script src="js/jquery.easytabs.js" type="text/javascript"></script>
  <script type="text/javascript">
    $(document).ready( function() {
      $('#tab-container').easytabs();
    });
  </script>
  <link rel="stylesheet" type="text/css" href="css/main.css">
</head>

<body>
  <div id="tab-container" class="tab-container">
    <ul class='etabs'>
      <li class='tab'><a href="#matchLinkHrefToDivId1">Tab 1</a></li>
      <li class='tab'><a href="#matchLinkHrefToDivId2">Tab 2</a></li>
      <li class='tab'><a href="#matchLinkHrefToDivId3">Tab 3</a></li>
    </ul>
    <div class='panel-container'>
    <div id="matchLinkHrefToDivId1">
      <h2>Tab 1</h2>
      <!-- content -->
    </div>
    <div id="matchLinkHrefToDivId2">
      <h2>Tab 2</h2>
      <!-- content -->
    </div>
    <div id="matchLinkHrefToDivId3">
      <h2>Tab 3</h2>
      <!-- content -->
    </div>
  </div>
</body>
</html>

How to implement EasyTabs.js

 
						
<ul class='etabs'>
    <li class='tab'><a href="#matchLinkHrefToDivId1">Tab 1</a></li>
    <li class='tab'><a href="#matchLinkHrefToDivId2">Tab 2</a></li>
    <li class='tab'><a href="#matchLinkHrefToDivId3">Tab 3</a></li>
</ul>
  1. Create a unordered list with a class of 'etabs'.
    2.
  2. Give a class of 'tab' to each list item added.
    3.
  3. Give each list item an anchor tag and set the anchor tags href to whatever you like as long as there is a # infront of it.
    4.

How to implement EasyTabs.js

 
						
<div class='panel-container'>
    <div id="matchLinkHrefToDivId1">
      <h2>Tab 1</h2>
      <!-- content -->
    </div>
    <div id="matchLinkHrefToDivId2">
      <h2>Tab 2</h2>
      <!-- content -->
    </div>
    <div id="matchLinkHrefToDivId3">
      <h2>Tab 3</h2>
      <!-- content -->
    </div>
</div>
  1. Create a div with a class name of 'panel-container'.
    2.
  2. Inside the 'panel-container' div add enough divs to match the number of list items/tabs created before.
    3.
  3. Give each div an id that corresponds with the anchor tag href you wish to link with that tab.
    4.
  4. Put the content you wish to appear for that tab in the div.
    5.

How to style EasyTabs.js

Styling EasyTabs.js

						
.etabs { 
	margin: 0; 
	padding: 0; 
}

.tab { 
	display: inline-block; 
	zoom:1; *display:inline; 
	background: #935135; 
	border: solid 1px #D4CDAF; 
	border-bottom: none; 
	-moz-border-radius: 4px 4px 0 0; 
	-webkit-border-radius: 4px 4px 0 0; 
}

.tab a { 
	font-size: 16px; 
	line-height: 2em; 
	display: block; 
	padding: 0 20px; 
	outline: none; 
	text-decoration: none;
	color:#D1D774;
	font-family: 'Helvetica Neue';
}


.tab a:hover { 
	text-decoration: underline;
}

.tab.active { 
	background: #D4CDAF; 
	padding-top: 6px; 
	position: relative; 
	top: 1px; 
	border-color: #D4CDAF; 
}

.tab a.active { 
	font-weight: bold; 
	color: #935135;
}
.tab-container .panel-container { 
	background: #D4CDAF; 
	border: solid #D4CDAF 1px; 
	padding: 20px; 
	-moz-border-radius: 0 4px 4px 4px; 
	-webkit-border-radius: 0 4px 4px 4px; 
	font-family: 'Helvetica Neue';
	color: #935135;

}

Styling EasyTabs.js is the same as styling any other page. Using class names, id names, tag names etc.. the page can be styled to the designers desire.
